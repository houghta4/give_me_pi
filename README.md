# Give me Pi
This script monitors online products to check if it is back in stock. 

I am using it to get a RPi 4 Model B at market price

## Supported domains (so far):
- Adafruit
- Pishop
- Vilros
- Canakit
- Chicago dist


## Required

- pip install twilio
- pip install python-dotenv
- pip install beautifulsoup4
